| 種類      | 演算子例                   | 例                 |
| ------- | ---------------------- | ----------------- |
| 算術演算子   | `+ - * / // % **`      | `a + b`, `a ** 2` |
| 比較演算子   | `== != > < >= <=`      | `a == b`          |
| 論理演算子   | `and or not`           | `a and b`         |
| 代入演算子   | `= += -= *= /= //= %=` | `x += 1`          |
| ビット演算子  | `&`                     | `^ ~ << >>`        |
| メンバー演算子 | `in not in`            | `"a" in "apple"`  |
| 同一性演算子  | `is is not`            | `a is b`          |
