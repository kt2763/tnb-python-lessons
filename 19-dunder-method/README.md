# #19 特殊メソッド

動画: https://youtu.be/OHXvBFT4Ewk

`__init__`, `__str__`, `__eq__`, `__iter__` などの特殊メソッド（dunderメソッド）を使ったクラスの拡張を扱います。

- `lesson19_1.py` — `__init__`（コンストラクタ）
- `lesson19_10.py` — ダメージ計算クラスでの特殊メソッド活用
- `lesson19_11.py` — ゲームセッション管理クラス
- `lesson19_12.py` — `__iter__` / `__next__`（イテレータの自作）
- `lesson19_13.py` — インベントリクラス（`__len__` など）
- `lesson19_14.py` — モンスタークラス（特殊メソッド総合例）
- その他 `lesson19_2.py` 〜 `lesson19_9.py` — 各種特殊メソッドの例
