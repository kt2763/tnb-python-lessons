# #3 エスケープシーケンス, f文字列

動画: https://youtu.be/b7M0CXCX7SE

エスケープシーケンス（`\n` `\t` `\\` など）、f文字列による変数埋め込みとフォーマット指定（小数点桁数、桁区切り、ゼロ埋め）、`str.format()` との比較を扱います。

- `lesson3_1.py` — エスケープシーケンス（改行、タブ、バックスラッシュ、クォート）
- `lesson3_2.py` — f文字列による変数埋め込みとフォーマット指定
- `lesson3_3.py` — `str.format()` による文字列フォーマット
- `lesson3_4.py` — 桁区切り・ゼロ埋めなどの高度なフォーマット指定
- `main.py` — サンプルのエントリーポイント
