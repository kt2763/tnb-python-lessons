# #18 標準ライブラリ

動画: https://youtu.be/KLQSjPlIAo4

`random`, `collections`, `itertools`, `pathlib`, `re` など、よく使う標準ライブラリの使い方を扱います。

- `lesson18_10.py` — `collections.defaultdict`
- `lesson18_11.py` — `itertools.combinations` / `permutations` / `product`
- `lesson18_12.py` — `itertools.count` / `cycle` / `repeat`
- `lesson18_13.py` — `pathlib.Path`（パス操作）
- `lesson18_14.py` — `re`（正規表現の基本）
- その他 `lesson18_2.py` 〜 `lesson18_16.py` — 標準ライブラリの各種活用例
