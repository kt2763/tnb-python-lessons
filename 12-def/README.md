`def` による関数定義、戻り値、デフォルト引数、可変長引数（`*args` / `**kwargs`）を扱います。

- `lesson12_2.py` — 戻り値のある関数

- `lesson12_5.py` — 可変長引数（`*args`）
- `lesson12_6.py` — キーワード可変長引数（`**kwargs`）
