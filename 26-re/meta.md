| 記号     | 意味                             | 例                    |
| -------- | -------------------------------- | --------------------- |
| `.`      | 任意の1文字（改行除く）          | `a.c` → abc, aXc      |
| `\d`     | 数字（0〜9）                     | `\d\d` → 12, 99       |
| `\w`     | 英数字 + アンダースコア          | `\w+` → hello, user_1 |
| `\s`     | 空白文字（スペース、タブ、改行） | `\s+` → 空白の連続    |
| `\D`     | 数字以外                         | `\D+` → abc, !!       |
| `[abc]`  | a, b, c のいずれか1文字          | `[aeiou]` → 母音      |
| `[a-z]`  | a〜z の1文字                     | `[a-zA-Z]` → 英字     |
| `[^abc]` | a, b, c 以外の1文字              | `[^0-9]` → 数字以外   |

| 記号    | 意味              |
| ------- | ----------------- |
| `*`     | 0回以上の繰り返し |
| `+`     | 1回以上の繰り返し |
| `?`     | 0回または1回      |
| `{n}`   | ちょうどn回       |
| `{n,m}` | n回以上m回以下    |

| 記号 | 意味         |
| ---- | ------------ |
| `^`  | 文字列の先頭 |
| `$`  | 文字列の末尾 |
| `\b` | 単語の境界   |

| フラグ                          | 意味                                  |
| ------------------------------- | ------------------------------------- |
| `re.IGNORECASE` (または `re.I`) | 大文字小文字を区別しない              |
| `re.MULTILINE` (または `re.M`)  | `^` と `$` が各行の先頭・末尾にマッチ |
| `re.DOTALL` (または `re.S`)     | `.` が改行にもマッチ                  |
