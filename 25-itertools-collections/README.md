# #25 itertools, collections

動画: https://youtu.be/vM7GboR2G0g

標準ライブラリ `itertools`（`chain` / `islice` / `cycle` / `product` / `combinations` / `groupby` など）と `collections`（`Counter` / `defaultdict` / `deque` / `namedtuple`）による効率的なデータ処理を扱います。

- `lesson25_1.py` — `itertools.chain` で複数のイテラブルを連結する
- `lesson25_2.py` — `itertools.islice` でイテラブルの一部を取り出す
- `lesson25_3.py` — `itertools.cycle` / `repeat` で繰り返す
- `lesson25_4.py` — `itertools.product` で組み合わせの全パターンを作る
- `lesson25_5.py` — `itertools.combinations` / `permutations` で組み合わせ・順列を作る
- `lesson25_6.py` — `itertools.groupby` でグループ化する
- `lesson25_7.py` — `collections.Counter` で要素の出現回数を数える
- `lesson25_8.py` — `collections.defaultdict` でキー未存在時の自動初期化
- `lesson25_9.py` — `collections.deque` による両端キュー・スライディングウィンドウ
- `lesson25_10.py` — `collections.namedtuple` で名前付きタプルを作る
