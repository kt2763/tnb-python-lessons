# #8 print()の便利テクニック

動画: https://youtu.be/fZrONvh9A0k

`print()` の `end` / `sep` 引数、書式指定、ファイルへの出力、`flush` など便利な使い方を扱います。

- `lesson8_1.py` — printの基本
- `lesson8_2.py` — `end` 引数（改行の制御）
- `lesson8_3.py` — `sep` 引数（区切り文字の変更）
- `lesson8_4.py` — f文字列での桁数指定
- `lesson8_5.py` — `.format()` との組み合わせ
- `lesson8_6.py` — ファイルへの出力（`file` 引数）
- `lesson8_7.py` — `flush` によるリアルタイム出力
